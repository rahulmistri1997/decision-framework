Framework Documentation

The Problem

Analysis paralysis. We often apply rigorous, time-consuming evaluation processes to low-stakes personal choices. Cost of Latency > Suboptimal Choices.

Phase 1: 10/10/10 Filter

Evaluates the choice across three time horizons to prevent "focalism" (inflating current importance):

  • 10 Minutes: Immediate emotional reaction.
  • 10 Months: Medium-term consequences.
  • 10 Years: Long-term memory/impact.

Phase 2: The Circuit Breaker

Determines reversibility. A "Two-Way Door" means you can easily roll back the decision. Reversible choices automatically downgrade the stakes severity, encouraging faster execution.

Output: Execution Tiers

🟢 Low Stakes (15m TTL) Logistics, minor tools. Pick the first viable option. Flip a coin if needed.
🟡 Medium Stakes (2h TTL) Mid-tier purchases, travel. Define 3 must-haves. Buy the first thing that hits all 3.
🔴 High Stakes (7d TTL) Investments, career. Set a fallback option today. If 7 days pass with no breakthrough, execute the fallback.

Decision Tree Logic

[ START ] 10/10/10 Impact Assessment
Both 10-Mo & 10-Yr = Zero
➔ Base: LOW STAKES
Either 10-Mo or 10-Yr = Major/Altering
➔ Base: HIGH STAKES
All other combinations
➔ Base: MEDIUM STAKES
[ CONTINUE ] Reversibility Check
Two-Way Door (Reversible)
➔ Downgrade Stakes by 1 Tier
(e.g., High becomes Medium)
One-Way Door (Irreversible)
➔ Maintain Stakes / Finalize
